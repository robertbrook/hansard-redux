This site has been produced from digitised editions of Commons and Lords Hansard, the Official Report of debates in Parliament. This was part of a project led by the Commons and Lords libraries.

There's the Hansard text itself by volume, Lords sittings, Commons sittings and Westminster Hall sittings as well as Written Answers, Written Statements, Lords reports and Grand Committee reports. This site also includes extracted lists of People who are recorded as having spoken, Constituencies, Offices, Acts, Bills and Divisions.

Please note that Parliament, as administrator of this website recently received notice from its hosting provider that the original hosting service would no longer be available from the end of October 2012. As such we have moved the content of the site to a new provider but have not yet been able to restore the search function. We are working hard on recreating the ability to search this material as a matter of priority.

If you have any questions or comments about this website, please contact webmaster@parliament.uk.

Information presented here is generated from the publicly available XML files. The source code used to create this website is available on Github or via the link on the credits page.
